# 133 프로젝트 · v1.2 (A/B 중앙모니터 설문 진행판)

## 무엇이 달라졌나요
- **가입하기 흐름을 PPT(14~16) 기획 그대로** 재구성했습니다:
  1) `가입하기 안내`(파란 카드) — 링크/설명/`비공식 서비스 가입` 버튼
  2) `가입 축하`(검정 카드) — `쿠폰받기` 버튼
  3) `쿠폰받기` 클릭 시 **중앙 모니터**에서 설문이 시작됩니다.
     - **홀수 문항 → 상단 A**, **짝수 문항 → 하단 B**
     - 각 문항은 **체크박스(다중 선택 가능)**, 하나라도 체크되면 **다음 문항이 반대 영역에 즉시 등장**
     - 마지막 문항 이후 **인사 문구** 출력

- **노란 컨트롤 도크**(B 하단 중앙)는 그대로 유지(◀ ▶ ▲ ▼, 자동재생 ■).

- **프로모션 배너 슬라이드**: `/assets`에 `P1.*, P2.*, P3.*` / `P1-1.*, P2-1.*, P3-1.*` 업로드하면 자동 반영.

## 파일
- index.html / css/styles.css / js/app.js
- assets/
  - P1.*, P2.*, P3.*, P1-1.*, P2-1.*, P3-1.*
  - 쿠폰신청.mp4, 긴급쿠폰신청.mp4
  - logo.png (옵션)

## 메모
- 설문 문구는 제공하신 텍스트를 **원문 그대로** 사용했습니다.
- 체크박스 방식(4지선다 원칙)이나 일부 3지선다 문항도 그대로 지원합니다.
